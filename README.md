# BankSearch App - Enqura Information Systems - Summer Intern

## Technologies
- Angular 14
- .NET 6
- Entity Framework
- REST API
- MSSQL

## Using
- Run script.sql file and install database; on this directory: (localdb)\MSSQLLocalDB) or move database to your User folder.
